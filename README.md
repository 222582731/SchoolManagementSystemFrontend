# School Management System

A full-stack school management system designed to streamline academic administration and facilitate communication between students and teachers.

## Key Features
- Secure login & authentication  
- Role-based access (Student & Teacher)  
- Student-teacher messaging system  
- Attendance management  
- Grade management  
- Coursework upload & download (PDF)  
- Responsive user interface  

## Tech Stack
**Backend:** Java, Spring Boot, RESTful APIs, JSON  
**Frontend:** HTML, CSS, JavaScript  
**Database:** MySQL  

## System Overview
The platform enables:

### Students to:
- View attendance records  
- Track academic performance  
- Download coursework  
- Communicate with teachers  

### Teachers to:
- Upload coursework materials  
- Update attendance and grades  
- Send messages to students  

## Setup Instructions
1. Clone both frontend and backend repositories  
2. Configure database credentials in `application.properties`  
3. Run the Spring Boot backend  
4. Open the frontend project in your browser
